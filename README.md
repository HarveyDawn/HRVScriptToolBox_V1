# HRVScriptToolBox_V1

ScriptToolBox/
├─ PythonScripts/     # Python 脚本源码
│  ├─ FileTool/        # 文件处理工具（重命名、格式转换、批量整理）
│  ├─ Crawler/         # 爬虫脚本（数据抓取、网页解析、导出）
│  └─ WinApp/          # GUI 桌面程序（可视化小工具）
│
├─ BatchShell/         # 批处理 / Shell 脚本
│  ├─ Windows/         # .bat 脚本（一键清理、启动、快捷操作）
│  └─ Linux/           # .sh 脚本（服务管理、巡检、自动化）
│
├─ BuildConfig/        # 打包 & 环境配置
│  ├─ SpecFiles/       # PyInstaller 打包配置（.spec）
│  ├─ Icons/           # EXE 图标文件
│  └─ Requirements/    # 依赖清单（requirements.txt）
│
├─ Output/             # 输出产物
│  ├─ ExeFiles/        # 打包好的 EXE 程序
│  └─ Results/         # 脚本运行输出文件
│
└─ README.md           # 工具箱说明 & 使用文档
